# NgCampaignBuap
NgCampaignBuap es una aplicación estudiantil creada con Angular para gestionar y mostrar campañas de manera efectiva en la BUAP (Benemérita Universidad Autónoma de Puebla). Este proyecto sirve como plantilla para construir aplicaciones web escalables y mantenibles utilizando Angular.

## Features

- **Modular Architecture**: Organized components for better maintainability.
- **Responsive Design**: Optimized for various devices.
- **Environment-Specific Configurations**: Separate settings for development and production.
- **Customizable Components**: Easily extendable for specific use cases.

## Getting Started

To get started with the NgCampaignBuap application, follow these steps:

1. **Clone the repository:**
   ```bash
   git clone https://github.com/jddjp/NgCampaignBuap.git
   ```

2. **Navigate to the project directory:**
   ```bash
   cd NgCampaignBuap
   ```

3. **Install dependencies:**
   ```bash
   npm install
   ```

4. **Run the application:**
   ```bash
   ng serve
   ```

5. **Open your browser and navigate to:**
   ```bash
   http://localhost:4200
   ```
